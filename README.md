Небольшое GUI-приложение на Tkinter для:
- онлайн-записи ЭЭГ с Arduino (через COM/Serial),
- загрузки CSV файлов (в том числе drag&drop),
- анализа λ-ритма (4–6 Гц) и α-ритма (7–13 Гц),
- построения графиков (сигнал / PSD / λ-ритм / сравнение),
- экспорта PDF отчёта.

## Структура проекта
- `app.py` — главный файл: интерфейс, вкладки, запуск анализа, экспорт PDF
- `config.py` — настройки (частоты, цвета UI, шрифты, стиль графиков)
- `io_csv.py` — чтение CSV, поиск времени/каналов, оценка FS
- `signal_analysis.py` — фильтрация, мощности, выводы
- `plotting.py` — функции построения графиков (Figure для Tk/PDF)

## Установка зависимостей

Перед запуском необходимо установить библиотеки командой:

pip install -r requirements.txt
